# Projeto de Validação de Nome

Este projeto é uma aplicação simples que utiliza Node.js e Express para validar se um nome não está vazio. O objetivo é demonstrar como criar um servidor web básico e realizar validações simples de dados.

## Funcionalidades

- Validação de Nomes: O projeto oferece uma rota que aceita um parâmetro `nome` na URL e valida se o nome não está vazio.

## Objetivo

O objetivo principal deste projeto é fornecer um exemplo prático de como criar um servidor Node.js com o framework Express e realizar uma validação simples de dados. Isso pode ser útil como ponto de partida para projetos mais complexos ou como uma base de aprendizado para desenvolvedores iniciantes.

## Tecnologias Utilizadas

- Node.js: Plataforma de tempo de execução JavaScript.
- Express.js: Framework para construir aplicativos web com Node.js.

## Executando o Projeto

Siga estas etapas para executar o projeto em sua máquina:

1. Clone ou faça o download deste repositório.

2. Abra um terminal e navegue até o diretório do projeto.

3. Execute o seguinte comando para instalar as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor Node.js executando o seguinte comando:

```bash
   node index.js
```
